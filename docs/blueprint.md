# **App Name**: MedCare Assist

## Core Features:

- QR Code Scan & Load: Scan a QR code to quickly load patient medical records into the app.
- Simplified Patient Profile: Display patient information in a simple, card-based layout with clear icons.
- Medication Timeline: Show medication schedules in an easy-to-read timeline with color-coded medicine shapes.
- Voice Assistant for Medication: Reads medication names, dosages, and timings aloud in the patient's preferred language.
- AI Symptom Checker: Allow the patient to speak or type symptoms and uses an AI tool to analyze the symptoms, show condition summary, and suggest next steps.
- Profile data save: Store all of the profile data in Firestore, so a new profile does not have to be created each time.

## Style Guidelines:

- Primary color: Soft medical blue (#3BAFDA) for a calm and trustworthy feel.
- Background color: Light blue (#E7F7FF) for a clean, medical aesthetic.
- Accent color: Teal (#87CEEB) for highlighting important information and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, suitable for headlines or body text
- Use large, minimal icons to represent key actions and information; focus on clarity for elderly users.
- Card-based layout with rounded corners to provide a friendly user interface with no clutter.
- Use smooth, modern animations to show progression, but maintain subtlety so as not to overwhelm.
